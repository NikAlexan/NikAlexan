# Минимальный style guide

Дата: 2026-01-30

## Типографика
- Заголовки: использовать один масштаб, избегать разнобоя в размерах.
- Основной текст: `text-base` или `text-sm` в зависимости от блока.
- Прозрачность текста — только через утилиты `.text-strong`, `.text-secondary`, `.text-muted`, `.text-disabled`.

## Отступы
- Базовая сетка: шаг 4/8 px (Tailwind spacing scale).
- Межсекционные отступы должны быть кратны 16/24/32.

## Цвета
- Все цвета задаются через CSS-переменные в `src/styles/global.css`.
- Не использовать `color-mix` и inline-значения в компонентах.

## Ссылки и кнопки
- Ссылки: всегда видимы в контексте (контраст + hover).
- Кнопки: один главный стиль (primary) и один вторичный (ghost/outline).

## Темы
- Переключение через класс `dark` на `html`.
- Нельзя использовать одновременно `data-theme` и `class`.

## Изображения
- Для больших изображений — WebP/AVIF через Astro Image (если появятся растровые кейсы).
- SVG иконки допускаются без оптимизации.
